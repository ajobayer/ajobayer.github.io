    ABOUT THE PROJECT

    S60 Calendar To-Do Management System :

  • In the S60 device, there is a built-in calendar which has a limited support from the To-Do or task management point of view. And S60 have no stand alone application to manage and track To-Do of day by day. Main features of this application are to-do list grouping, priority setting, localization and user interactive text search. The next version enriches features like encryption facility implement calendar view and synchronization with To-do calendar entry. .

    • S60 DriveCleanser :

  • The product can clean drives of S60 phones in a way so that no deleted data can be recoverable using conventional data recovery software. This was applicable for both S60 MMC and Phone memory.

    • S60 RemoteLock (Mobile Theft Protector) :

  • Secure the specific data like SMS, contact number, image, video documents, sound files, and calendar items when mobile phone will be theft or lost. Application is function like when SIM is changed application will automatically lock the phone and send a notification SMS to predefined mobile number. This will automatically detect the command SMS to do specific task from remote. Mobile Theft Protector is will be run permanently in background on mobile phone
